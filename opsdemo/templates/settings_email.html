Which should I use?

📡
SMTP

Works locally and on paid Render plans. Free tier Render blocks ports 465/587 so you'll get "Network unreachable" errors there.

⚡
SendGrid API

Free: 100 emails/day. Sign up at sendgrid.com → API Keys → create key with "Mail Send" permission. Username for SMTP relay is literally apikey.

✉
Resend API

Free: 3,000 emails/month. Sign up at resend.com → add & verify your domain → create API key. From address must use your verified domain.

🔒
Credentials stored encrypted

Both SMTP passwords and API keys are encrypted at rest with your CREDENTIAL_ENCRYPTION_KEY.
