How it works

✉
Invoice emails

On the Renewals page, click Send Invoice next to any renewal to create an invoice and optionally email it to the contact.

🔒
Secure storage

Your SMTP password is encrypted at rest using the same key as integration credentials. Never stored in plain text.

⚡
Common providers

Works with Gmail (use an App Password), Outlook, SendGrid, Mailgun, Postmark, or any SMTP relay. Port 587 + STARTTLS is recommended.
